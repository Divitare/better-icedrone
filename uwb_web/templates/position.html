Position Estimation

Phase 2 — Coming Soon

This page will provide real-time 2D/3D position estimation using trilateration from the range measurements collected by the UWB anchors.

Prerequisites

  • At least 3 anchors for 2D, 4 for 3D positioning.
  • Anchor coordinates must be configured on the Devices page (X, Y, Z fields).
  • Anchors must be marked with the Anchor checkbox.

Planned Features

  • Real-time 2D scatter plot of tag position
  • 3D position estimation via least-squares trilateration
  • Kalman filter for smooth tracking
  • Configurable outlier rejection
  • Position history trail
  • CSV export of position estimates
